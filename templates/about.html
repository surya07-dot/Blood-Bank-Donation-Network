About BloodCare

BloodCare is a mini project built using Python (Flask) with PostgreSQL database and a modern responsive front-end using HTML, CSS and JavaScript.

  • Donor registration with contact & blood group details.
  • Real-time blood stock overview.
  • Emergency blood request form.
  • Admin dashboard for staff with login & role-based actions.

You can run this project in PyCharm by installing the requirements and configuring the PostgreSQL connection in config.py.
